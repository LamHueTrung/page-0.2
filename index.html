Lý Thuyết

Để có 1 bố cục chỉnh chu, ưa nhìn các bạn cần có thể trình bày như bố cục sau: 
                        
int main() {
    code1;
    code2;
    ....
    return 0;
}
Bắt đầu các bạn thục vào 1 khoảng (tab) và bắt đầu viết code. Lưu ý: kết thúc 1 dòng code phải có dấu ;. khi viết code lưu ý đặt dấu {} đúng vị trí để khi nhìn vào ta biết hàm bắt đầu và kết thúc tại đâu. Tương tự: Nếu trong hàm có thêm 1 hàm khác thì các dòng code trong hàm đó nên thục vào 1 khoãng để khi nhìn vào ta biết các dòng code đó nằm trong hàm 
                        
int main() {
    code1;
    code2;
    ....
    funcion {
        code1;
        ....
        code n;
    }
    return 0;
}
Đọc tới đây bạn đã biết cách bố trí bố cục khi viết code sao cho hợp lí rồi đúng không ? Chúc các bạn thành công.

Cách sắp xếp bố cục khi viết code

Nhắc nhở: Đây là 1 số lưu ý khi các bạn viết code.
